Сделано на golang generic + gin, реализованы только CRUD-методы работы с объектами User и Link.

Доступный функционал:

Users - Пользователи

Модель данных 

type User struct {
ID         int    `json:"id,omitempty" mapstructure:"id"`
Username   string `json:"username,omitempty" mapstructure:"username" form:"username"`
Password   string `json:"password,omitempty" mapstructure:"password" form:"password"`
FirstName  string `json:"first_name,omitempty" mapstructure:"first_name"`
LastName   string `json:"last_name,omitempty" mapstructure:"last_name"`
Email      string `json:"email,omitempty" mapstructure:"email"`
Phone      string `json:"phone,omitempty" mapstructure:"phone"`
UserStatus bool   `json:"user_status,omitempty" mapstructure:"user_status"`
}

Методы (path /api/users/)

  • GET - Получить данные пользователя

    curl -X GET http://localhost:8080/api/users/5

  • POST - Создать нового пользователя

    curl -X POST http://localhost:8080/api/users/ -H 'Content-Type: application/json' -d '{"username":"tester", "password":"test", "first_name":"First", "last_name":"Last", "email":"tester@example.loc", "phone":"222"}'

  • PUT - Обновить данные пользователя

    curl -X PUT http://localhost:8080/api/users/ -H 'Content-Type: application/json' -d '{"id":7, "username":"Updated", "user_status":true}'

  • DELETE - Удалить пользователя

    curl -X DELETE http://localhost:8080/api/users/5

Links - ссылки

Модель данных 

type Link struct {
ID           int    `json:"id,omitempty" mapstructure:"id"`
LongLink     string `json:"long_link,omitempty" mapstructure:"long_link"`
ClickCounter int    `json:"click_counter,omitempty" mapstructure:"click_counter"`
OwnerID      int    `json:"owner_id,omitempty" mapstructure:"owner_id"`
IsActive     bool   `json:"is_active,omitempty" mapstructure:"is_active"`
}

Методы (path /api/links/)

  • GET - Получить ссылку по ID

    curl -X GET http://localhost:8080/api/links/7

  • POST - Создать новую ссылку

    curl -X POST http://localhost:8080/api/links/ -H 'Content-Type: application/json' -d '{"long_link":"http://r0.ru"}'

  • PUT - Обновить ссылку

    curl -X PUT http://localhost:8080/api/links/ -H 'Content-Type: application/json' -d '{"id":5, "long_link":"http://r0.ru", "is_active":true}'

  • DELETE - Удалить ссылку

    curl -X DELETE http://localhost:8080/api/links/7
